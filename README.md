# so-escalonador

Projeto realizado para a matéria de **Sistemas Operacionais** da UTFPR.

Para a compilação do programa, execute o seguinte comando:

> make

Caso queira executá-lo logo em seguida, execute:

> make run

É necessário ter o gcc instalado, assim como ter acesso à biblioteca ncurses.

## Objetivo
O intuito desse trabalho é simular a execução de um **Sistema operacional multitarefa monoprocessado preemptivo** com diferentes configurações:
- Algoritmo de escalonamento
- Quantum do sistema
- Tarefas a serem executadas
